Interactive Games and genre Design Comparisons.

The website analyses the different genres of games, with the selection of them being FPS, RPG, Sandbox, and Gacha. As the website will compare the different games with each other from different or same genre. Highlighting the differences in how the games approach the gameplay, mechanics, graphics, story and design in the games.

Game Genre

FPS

First person shooter ranging from fast to slow and heavily using game sense to prove you skill in these games.

RPG

A role playing story games with wide and vast world for players to explore and loot.

Sandbox

A creative type of genre allowing players to fully use the games limits to build, explore and do whatever they basically want.

Gacha

A wide open game with many genres but its most noticable one being Gacha having its own unique anime type style and having a character collection with random reward system.

About This Project

Developed for COMP1004 Computing Practice using HTML, CSS, and JavaScript. As the aim of the website is to analyse and compare the differences and similarites of games, and focusing on how the gameplay, design choices, and player interactions vary between them. Other aims of the website is to give the user a clear view on each game to give them ideas of how they are played out giving them small ideas.

The website a interacitve navigation, dynamic content updates with JavaScript and responsive desigfn principles to create an engaging user experience.
